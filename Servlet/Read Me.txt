How to Execute

1.Place "LoginServlet.Java" under

"C:\Program Files\Apache Software Foundation\Tomcat 7.0\webapps\ROOT\WEB-INF\classes"

2. Execute LoginServlet.java using jGrasp(which generates LoginServlet.class in the same directory as given in step-1)

3. Place "pub_login.html" under 
"C:\Program Files\Apache Software Foundation\Tomcat 7.0\webapps\ROOT"

4. Open the Web.xml which is present under "C:\Program Files\Apache Software Foundation\Tomcat 7.0\webapps\ROOT\WEB-INF" and add servlet name and servlet mapping name as given in the web.xml file given in this folder  

4. Start Apache Server

5. Run "pub_login.html" on Apache server as below

"http://localhost:8080/pub_login.html"
